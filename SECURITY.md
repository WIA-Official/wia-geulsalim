# Security Policy · 보안 정책

## 지원 버전 · Supported Versions

| Version | Status |
|---------|--------|
| main (latest) | ✅ 보안 업데이트 지원 |

## 취약점 신고 · Reporting a Vulnerability

WIA 글살림의 보안 취약점을 발견하셨다면, 공개 이슈로 등록하지 마시고 아래 경로로 비공개 제보 부탁드립니다.

- 📧 **Email**: `global@thekoreantoday.com`
- 🔒 **GitHub Security Advisory**: [Report privately](https://github.com/WIA-Official/wia-geulsalim/security/advisories/new)

## 응답 SLA · Response Commitment

| Action | Timeline |
|--------|----------|
| 최초 수신 확인 (Acknowledge) | **48시간 이내** |
| 취약점 검증 (Triage) | **7일 이내** |
| 패치 배포 (Fix) | **30일 이내** (중대 취약점: 7일 이내) |
| 공개 disclosure | 패치 배포 후 **30일 뒤** |

## 보안 원칙 · Principles

- **Principle of Least Privilege** — 최소 권한 원칙
- **Defense in Depth** — 계층적 방어
- **Secure by Default** — 기본값이 안전
- **Transparent Fixes** — 모든 보안 패치는 변경 내역 공개

## 적용 보안 표준 · Security Standards

- HTTPS 전면 적용 (HSTS · SSL Labs A+ 등급)
- Content-Security-Policy · X-Content-Type-Options · X-Frame-Options
- GDPR · PIPA(한국 개인정보보호법) 준수
- OWASP Top 10 대응
