2. 기술 명세

2.1 기술목적

훼손·언어·포맷·감각 4중 장벽이 인류 5천년 지식의 95%를 가두고 있다. ①한국 고문서 87.2%(451,304건)는 미디지털화, ②세계 7,000개 언어 중 97%는 AI 품질 현격 미달, ③공개 PDF 80%+는 접근성 태그 없음, ④3.2억 감각장애인은 글에 닿을 수 없다.

WIA 글살림은 이 4개 장벽을 단일 AI 파이프라인으로 동시 해결한다. EU EAA(2025.6.28 강제시행), 한국 장애인차별금지법, 미국 ADA·Section 508 등 글로벌 법적 의무화 환경에서, 기존 도구로는 해결 불가한 통합 접근성 AI의 부재를 채운다.

  • 고문서 자동 복원: Diffusion 이미지 복원 + LLM 텍스트 보정 · 원본 해시 보존 (복원 ≠ 조작)
  • 접근성 자동 구조화: WCAG 2.2 AA · EPUB A11y 1.1 · KS X 6201 · EU EAA 4개 국제표준 동시 대응 · 80개 항목 자동 검사 (a11y.wiabook.com)
  • 다국어 점자·음성 변환: WIA Braille 7,000개 언어 · IPA 143MB DB · TTS 254개 언어
  • 3D 수어·WebXR 전달: WIA Talk 3D 아바타 수어 · wia.tools 436개 대화형 도구 · Meta Quest / iPhone AR
  • 통합 파이프라인 오케스트레이션: 한 번 업로드로 4단계 자동 처리 · 무인 실행

2.6 혁신적 요소

  • 복원·구조화·변환·전달 4단계 단일 파이프라인 통합 (기존 도구는 단계별 개별 처리 — 통합 상용 플랫폼 현재 미확인)
  • 7,000개 언어 점자 + 254개 언어 음성 동시 지원 (동일 기능 통합 상용 플랫폼 현재 미확인)
  • 원본 해시 보존 기반 복원 무결성 보장 (복원 ≠ 조작 원칙 · 딥페이크 차단)

2.7 도전적 요소

Diffusion 이미지 복원 + LLM 텍스트 추론 하이브리드로 훼손 고문서를 복원하는 범용 엔진은 현재 기술 수준 대비 높은 복합성. 대회 기간 핵심 과제: ①WIHP-AI를 30개→7,000개 언어로 확장하는 멀티링구얼 LLM 파인튜닝, ②WIA Talk 3D 수어 LLM 시퀀스 학습, ③Diffusion 복원 엔진 GPU 학습.

3.1 구현범위

세부업무 #1: 문서 복원 엔진 (신규 30%)

  • 새로운 패러다임 제시: 복원·구조화·변환·전달 4단계 통합 AI 파이프라인 — 훼손 글을 모든 감각·모든 언어로 살려내는 범용 아키텍처. 기존 단계별 도구의 사일로를 제거.
  • 특허 및 학술 가능성: 출원 3건 (WebXR 독서 시스템 · .wiabook 포맷 · 개인사진 VR 환경). 고문서 복원 Diffusion+LLM 하이브리드, 7,000개 언어 점자 알고리즘 논문 가능.
