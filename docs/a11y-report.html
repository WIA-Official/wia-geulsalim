감사 엔진 2 · Google Lighthouse (통합 실행)

구글 공식 웹 품질 감사 도구 · 4대 카테고리 · a11y.wiabook.com 백엔드 통합
Google이 제공하는 표준 웹 품질 감사 엔진으로, 전 세계 웹 개발 품질 기준의 사실상 표준입니다. Performance · Accessibility · Best Practices · SEO 4대 영역을 평가하며, Core Web Vitals(Google 검색 순위 반영)를 포함합니다. a11y.wiabook.com은 이 엔진을 직접 통합하여 하나의 감사 요청으로 axe-core와 Lighthouse를 동시 실행하고 결과를 병합 반환합니다.
